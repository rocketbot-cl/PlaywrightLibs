# PlaywrightLibs

Repository dedicated to storing the libraries required for the Playwright module for different operating systems, in order to facilitate its installation.

## Dependencies used in windows:
- playwright==1.58.0
- greenlet==3.3.1
- pyee==13.0.0
- typing_extensions==4.15.0

## Dependencies used in macos:
- playwright==1.58.0
- greenlet==3.3.2
- pyee==13.0.1
- typing_extensions==4.15.0
